Lytelux — Landing Page
======================

This is a minimal static landing page for Lytelux. The goal is to provide a clean placeholder while we rebuild the site from scratch.

Structure
- `index.html`: Main landing page markup.
- `styles.css`: Simple, responsive styles with light/dark support.

Local Preview
- Open `index.html` directly in a browser, or serve locally:
  - Python: `python3 -m http.server -d . 8080` and visit `http://localhost:8080/`
  - Node (if installed): `npx serve .` then follow the URL

Customization
- Update text and links in `index.html`.
- Tweak colors and spacing in `styles.css`.
- Favicon and logo use inline SVG; replace with brand assets when ready.

Notes
- This page avoids external fonts/scripts for speed and simplicity.
- Basic Open Graph tags are included for social sharing.
